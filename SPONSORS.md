## ❓ 常见问题

**Q: 我需要 GitHub 账户才能赞助吗？**
A: GitHub Sponsors 需要 GitHub 账户。你也可以通过其他方式支持（Star、分享、贡献）。

**Q: 赞助的钱会怎么用？**
A: 主要用于服务器费用、域名续费、宣传推广、开发者激励。所有支出公开透明。

**Q: 项目会变成付费的吗？**
A: 不会！LegalEase 永远 100% 开源免费。赞助只是帮助项目可持续发展。

**Q: 我可以一次性赞助吗？**
A: 可以！GitHub Sponsors 支持月付和一次性付款。
