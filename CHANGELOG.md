# 更新日志

所有重要的 LegalEase 更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.2.0] - 2024-03-25

### ✨ 新增功能

#### 多模型支持
- 新增 OpenAI GPT-4 支持
- 新增 Anthropic Claude 支持 (claude-3 系列)
- 新增本地 Ollama 支持 (完全离线)
- 统一的模型管理界面 (设置面板)
- 各提供商独立配置 (API Key、模型名称)

#### 法律计算器
- 工伤赔偿计算器 (10级伤残系数)
- 诉讼费计算器 (按标的额阶梯计费)
- 支持保留劳动关系选项
- 实时计算结果展示
- 法律依据标注

#### 设置系统
- 完整的设置面板 (模型、主题、语言、字体)
- 设置持久化 (localStorage)
- 自动加载上次配置
- 一键重置功能

#### UI/UX 改进
- 聊天界面显示当前模型指示器
- 来源引用优化 (法条/案例标签)
- 响应式设计优化
- 加载状态提示

### 🐛 Bug 修复
- 修复演示模式响应不完整问题
- 修复来源显示逻辑
- 优化错误处理

### 🔧 技术改进
- 引入 Zustand 状态管理
- 重构 API 路由 (支持多提供商)
- 优化嵌入模型调用
- 改进错误边界处理

### 📚 文档
- 新增 ROADMAP.md (开发路线图)
- 更新 README.md (突出新功能)
- 更新 ANDROID_BUILD.md (Android 构建指南)
- 新增 QUICKREF.md (快速参考)
