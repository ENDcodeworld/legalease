# LegalEase 开发路线图

## 🎯 当前目标：1000+ GitHub Stars

### 阶段一：核心功能完善（本周）
- [x] 基础 RAG 问答系统
- [x] Android APK 支持
- [x] Docker 一键部署
- [ ] **用户认证系统** - 保存聊天历史
- [ ] **法条收藏功能** - 用户可收藏常用法条
- [ ] **搜索历史** - 记录用户查询
- [ ] **多模型支持** - Claude、Ollama 等

### 阶段二：数据丰富（2周内）
- [ ] 导入完整民法典（1260条）
- [ ] 导入劳动法、劳动合同法
- [ ] 导入刑法、刑事诉讼法
- [ ] 导入交通事故赔偿标准
- [ ] 添加 1000+ 真实案例
- [ ] 建立案例标签体系

### 阶段三：用户体验（3周内）
- [ ] **法律计算器** - 诉讼费、工伤赔偿、人身损害赔偿
- [ ] **文书生成器** - 起诉状、答辩状模板
- [ ] **语音输入** - 支持语音提问
- [ ] **夜间模式** - 护眼主题
- [ ] **多语言** - 英文版支持
- [ ] **PWA 离线** - 缓存常用法条

### 阶段四：生态扩展（1个月内）
- [ ] **微信小程序** - 触达更多用户
- [ ] **Chrome 插件** - 网页划词查询
- [ ] **API 开放** - 允许第三方集成
- [ ] **数据导出** - 支持导出聊天记录
- [ ] **团队协作** - 多用户共享数据

### 阶段五：增长黑客（持续）
- [ ] **推荐奖励** - 邀请好友获得更多查询次数
- [ ] **社区共建** - 用户可以贡献案例
- [ ] **排行榜** - 热门查询TOP100
- [ ] **法律日历** - 重要法律纪念日提醒
- [ ] **案例追踪** - 订阅感兴趣的案件进展
