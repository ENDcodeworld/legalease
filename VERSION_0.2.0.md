## 1. 🤖 多模型支持

### 为什么需要多模型？

不同的 AI 模型各有优势：
- **GPT-4**: 知识全面、推理能力强、响应快
- **Claude**: 法律领域优化、长文本处理、安全性高
- **Ollama**: 数据隐私保护、完全离线、无限免费

### 如何使用？

1. 点击聊天页面右上角的 **⚙️ 设置** 按钮
2. 选择模型提供商：
   - **OpenAI** - 输入你的 API Key，选择模型版本
   - **Anthropic** - 输入你的 API Key，选择 Claude 版本
   - **Ollama** - 输入本地 Ollama 服务器地址 (默认 http://localhost:11434)

3. 设置会自动保存到浏览器，下次自动加载

### 注意事项

- **OpenAI**: 需要 [platform.openai.com](https://platform.openai.com) API Key
- **Anthropic**: 需要 [console.anthropic.com](https://console.anthropic.com) API Key
- **Ollama**: 需要本地安装并运行 Ollama，推荐模型：`llama2:13b-chat`、`mistral`、`codellama`

## 2. 📊 法律计算器

### 功能特性

LegalEase 内置专业法律计算器，帮你快速计算：

#### ✅ 工伤赔偿计算
根据《工伤保险条例》精确计算：
- 一次性伤残补助金 (1-10级系数)
- 停工留薪期工资
- 工伤医疗补助金
- 伤残就业补助金

**计算依据**：
- 伤残等级：1级（最重）到 10级（最轻）
- 月工资基数
- 是否保留劳动关系
- 年龄因素（用于就业补助金）

#### ✅ 诉讼费计算
根据《诉讼费用交纳办法》自动计算：
- 财产案件：按标的额比例收费
- 非财产案件：按件收费
- 阶梯费率：10,000 元 → 100万+ 不同档位

**使用示例**：
- 索赔 10 万元 → 诉讼费约 2,300 元
- 索赔 100 万元 → 诉讼费约 13,800 元

### 如何使用？

1. 在聊天页面点击 **"计算器"** 标签页
2. 选择计算器类型（工伤/诉讼费）
3. 填写相关参数
4. 点击 **"计算"** 按钮
5. 查看详细结果和法律依据

## 3. ⚙️ 设置面板

### 可配置项

| 设置项 | 选项 | 说明 |
|--------|------|------|
| **模型提供商** | OpenAI / Anthropic / Ollama | 选择 AI 模型 |
| **API Key** | 文本输入 | 对应提供商的 API 密钥 |
| **模型名称** | 下拉选择 | gpt-4, gpt-3.5-turbo, claude-3 等 |
| **主题** | 浅色 / 深色 / 自动 | 界面颜色主题 |
| **字体大小** | 小 / 中 / 大 | 控制文字大小 |
| **语言** | 中文 / English | 界面语言（暂未完全支持）|
| **显示来源** | 开/关 | 是否在回答中显示引用来源 |
| **保存历史** | 开/关 | 是否保存聊天记录到本地 |

### 数据隐私

- 所有设置仅保存在**你的浏览器本地存储**中
- 不会上传到任何服务器
- 更换设备需要重新配置
- 清除浏览器数据会丢失设置

## 🔧 更新你的部署

### 如果你已有部署：

1. **拉取最新代码**：
   ```bash
   git pull origin main
   ```

2. **更新依赖**（主要是新增 zustand）：
   ```bash
   npm install
   ```

3. **重新构建**：
   ```bash
   npm run build
   ```

4. **重启服务**：
   ```bash
   npm start
   # 或
   docker-compose restart
   ```

### 首次部署：

参考 README.md 快速开始章节，v0.2.0 包含所有新功能。

## 💡 使用技巧

### 1. 如何选择模型？
- **日常咨询**：GPT-4 Turbo (性价比高)
- **复杂分析**：Claude 3 Opus (推理能力强)
- **隐私敏感**：本地 Ollama (数据不外流)

### 2. 计算器使用建议
- 工伤赔偿：先咨询当地社保部门确认系数
- 诉讼费：仅作预估，法院可能有调整

### 3. 设置优化
- 开启"显示来源"可以查看引用的法条
- 调整字体大小提升阅读体验
- 定期导出聊天记录（暂未支持）
